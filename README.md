## 🎨 Deep Dive into the Tasks

### 🥤 Task 1: Pepsi Brand Landing Page
A visually stunning, high-fidelity brand landing page showcasing the iconic Pepsi flavors.
- **✨ Features:**
  - **Dynamic Theme Changer:** Click on different Pepsi cans/bottles to morph the page background color dynamically (Classic Blue, Wild Cherry Red, and Zero Sugar Black).
  - **Fluid Animations:** Smooth transitions and transitions for images and background gradient shifts.
  - **Social Integration:** Sleek footer with social media icon overlays.
  - **Responsive Design:** Fits seamlessly on desktop, tablet, and mobile displays.

---

### 🧮 Task 3: Glassmorphic Calculator
A sleek, elegant calculator with modern aesthetics and pixel-perfect functionality.
- **✨ Features:**
  - **Modern UI:** Built with a clean grid layout, subtle hover transitions, and readable numbers.
  - **Robust Engine:** Handles complex decimal inputs, continuous operations, all-clear (`AC`), and delete (`DEL`) functions.
  - **Feedback Effects:** Subtle micro-interactions on button clicks for a premium native app feel.

---

### 💱 Task 4: Real-Time Currency Converter
An efficient and dynamic currency converter powered by the robust, open-source **Frankfurter API**.
- **✨ Features:**
  - **Live Exchange Rates:** Fetching accurate, real-time market rates directly from the API.
  - **Intuitive UI:** Double selection system allows instantly converting between 30+ major global currencies (USD, EUR, GBP, JPY, etc.).
  - **Smart Validation:** Automatic prevention of redundant same-currency conversion with user-friendly alerts.

## 🚀 Getting Started & Local Setup

Want to run these projects on your local machine? No build steps or heavy dependencies are required—just open and play!

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/udayrastogi0531/TECHNOHACKS_EDUTECH-Web-Dev-Intern.git
cd TECHNOHACKS_EDUTECH-Web-Dev-Intern
```

### 2️⃣ Running a Task
You can open the projects in any modern browser:
- **Option A (Double-Click):** Navigate to the task directory (e.g., `Task3_Calculator_Web_Development`) and double-click `index.html`.
- **Option B (VS Code Live Server - Recommended):** Open the root folder in VS Code, right-click any `index.html` file, and select **"Open with Live Server"** to enjoy auto-reloads.
